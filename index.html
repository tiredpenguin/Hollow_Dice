Kiwi Kiwi

  • High Concept: Avoid your own worst enemy and last the longest.
  • Player Experience Goals: Experience being a kiwi.
  • Genre: Arcade
  • Game play: Basic movements with sound effects.
  • Prototype goal: Randomly generated elements at varying times.
  • Features: The randomly generated elements
  • Setting/Premise: A kiwi living in the world where if it is hit with its fruit counterpart it float into space.
  • Story: If the game has a story, summarize it in a paragraph. How will the dramatic tension interact with the gameplay tension?
  • Target Audience: Young audience
  • Play time: Very short
  • Strategies: Learning when the kiwis may come
  • Prototype Assets: Sounds are borrowed from examples. Time codes and random event code are also borrowed from some examples
  • Prototype Closest Other Game: Avoid incoming object games such as Temple Run
